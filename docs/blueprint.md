# **App Name**: CarVest

## Core Features:

- Car Investment Platform: Enable users to browse, select, and invest in car rental opportunities.
- Automated ROI Calculator: AI-powered tool to predict ROI based on car type, investment duration, and market trends.
- Referral System: Implement a multi-level referral system with unique referral links and real-time tracking.
- Withdrawal System: Allow users to withdraw earnings via mobile money and bank transfer with a minimum withdrawal amount of 100 GHS.
- Payment Integration: Integrate Flutterwave and Paystack for secure payment processing with GHS currency support.
- User Authentication: Implement secure user authentication with email/password, Google login, email verification, and password strength requirements.
- Admin Dashboard: Provide an admin dashboard for user management, investment approvals, and financial reporting.

## Style Guidelines:

- Primary color: Deep blue (#315ba1) to convey trust and stability in financial transactions.
- Background color: Light blue (#e0e8ef), desaturated to ensure the primary color is emphasized.
- Accent color: Purple (#8367bb) as it contrasts with the deep blue, signaling forward thinking and high quality. It can highlight key actions.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines and 'Inter' (sans-serif) for body text, creating a modern and readable interface.
- Use modern, minimalist icons to represent different investment options and features.
- Maintain a clean and responsive layout that is easy to navigate on both desktop and mobile devices.
- Incorporate subtle animations and transitions to enhance user engagement and provide feedback.
